# Student management system(java+Mysql)

##  Description
A Java-based application to manage student records using MySQL database with JDBC connectivity.

##  Features
- Add Student
- View Student Details
- Update Student Information
- Delete Student
- Login Authentication

## 🛠️ Technologies Used
- Java
- MySQL
- JDBC

##  Project Structure
- src → Java source code
- student_db.sql → Database script

##  How to Run
1. Import project into Eclipse
2. Run student_db.sql in MySQL Workbench
3. Execute Java files.
